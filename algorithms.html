Algorithms (10 Items)

1. Linear Search

Explanation: Checks each element in a list one-by-one until the target is found.

SET found TO false
FOR each element IN list
 IF element = target THEN
  found = true
END FOR
PRINT found
2. Binary Search

Explanation: Splits a sorted list in half repeatedly until the target is found.

WHILE low ≤ high
 mid = (low + high)/2
 IF arr[mid] = target THEN RETURN mid
 ELSE IF target < arr[mid] THEN high = mid - 1
 ELSE low = mid + 1
END WHILE
3. Bubble Sort

Explanation: Repeatedly swaps adjacent items to sort a list.

[4,3,1] → [3,1,4] → [1,3,4]
4. Selection Sort

Explanation: Finds the smallest value and moves it to the front repeatedly.

[5,2,8] → [2,5,8]
5. Insertion Sort

Explanation: Builds the sorted list one item at a time.

[4,2,3] → [2,3,4]
6. Merge Sort

Explanation: Divides the list into halves, sorts them, and merges.

[6,3,8] → split → merge → [3,6,8]
7. Quick Sort

Explanation: Picks a pivot and partitions list into smaller and larger parts.

pivot=5 → arrange → sort
8. Fibonacci Algorithm

Explanation: Each number is the sum of the two previous numbers.

0,1,1,2,3,5,8...
9. Factorial Algorithm

Explanation: Multiplies a number by every positive integer below it.

5! = 5×4×3×2×1 = 120
10. Palindrome Check

Explanation: Checks if a string reads the same forwards and backwards.

"level" → palindrome
