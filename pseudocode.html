10 Pseudocode Examples

1. Simple Addition
Explanation: Adds two numbers input by the user and outputs the sum.

INPUT num1, num2
SET sum = num1 + num2
OUTPUT sum
2. Finding the Largest of Three Numbers
Explanation: Compares three numbers and prints the largest one.

INPUT a, b, c
IF a > b AND a > c THEN
 OUTPUT a
ELSE IF b > c THEN
 OUTPUT b
ELSE
 OUTPUT c
END IF
3. Calculating Average (Fixed Count)
Explanation: Takes 5 numbers, sums them, and calculates the average.

SET total = 0
SET count = 0
FOR i FROM 1 TO 5 DO
 INPUT number
 SET total = total + number
 SET count = count + 1
END FOR
SET average = total / count
OUTPUT average
4. Check if Number is Positive/Negative/Zero
Explanation: Determines if a number is positive, negative, or zero.

INPUT num
IF num > 0 THEN
 OUTPUT "Positive"
ELSE IF num < 0 THEN
 OUTPUT "Negative"
ELSE
 OUTPUT "Zero"
END IF
5. Countdown Loop (DO-WHILE)
Explanation: Counts down from 10 to 1 using a loop that executes at least once.

SET counter = 10
DO
 OUTPUT counter
 SET counter = counter - 1
WHILE counter > 0
6. Sum of Numbers 1 to N (WHILE Loop)
Explanation: Computes the sum of numbers from 1 to N using a WHILE loop.

INPUT N
SET total = 0
SET i = 1
WHILE i <= N DO
 SET total = total + i
 SET i = i + 1
END WHILE
OUTPUT total
7. Greeting Multiple Times (FOR Loop)
Explanation: Prints "Hello!" three times using a FOR loop.

FOR count FROM 1 TO 3 DO
 OUTPUT "Hello!"
END FOR
8. User Input Validation (Loop until valid)
Explanation: Ensures the user enters a number between 0 and 100.

GET input_value
WHILE input_value < 0 OR input_value > 100 DO
 OUTPUT "Invalid input. Enter a number between 0 and 100."
 GET input_value
END WHILE
OUTPUT "Valid input received: " + input_value
9. Treasure Hunt Hint Logic
Explanation: Provides directions based on player and treasure positions.

// Assume treasure_row, treasure_col are set
// Assume player_row, player_col are inputs
IF player_row < treasure_row THEN
 OUTPUT "Go South"
ELSE IF player_row > treasure_row THEN
 OUTPUT "Go North"
ELSE
 IF player_col < treasure_col THEN
  OUTPUT "Go East"
 ELSE IF player_col > treasure_col THEN
  OUTPUT "Go West"
 ELSE
  OUTPUT "Found it!"
 END IF
END IF
